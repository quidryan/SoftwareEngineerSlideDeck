# Software Engineer Slide Deck

A short talk for middle school students about what it's like to be a Software Engineer.

**View the slides:** http://justin.halfempty.org/SoftwareEngineerSlideDeck/

## Requirements

- Node.js 20+ (use `nvm install 20 && nvm use 20` if needed)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
